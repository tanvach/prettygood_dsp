Prettygood DSP - WebUSB Configurator (Experimental)

Receiver JSON contains the current DSP settings. Try copy and paste the JSON in Sender and press enter. The DSP setting will update immediately. Press 'Save to flash' to save the setting permanently.

This tool is *experimental* - there are little checks, so invalid values may affect the DSP in unexpected ways. Reflash or updating the firmware will reset values to the built-in defaults.
